## How to use it

### 1. Open Cursor settings JSON

Cmd+Shift+P → “Preferences: Open User Settings (JSON)” (or Cursor Settings → open the JSON file).

### Merge the snippet into your settings

Copy the contents of gruvbox-dark-mix-settings-snippet.json and paste them into the root of your settings.json.
If you already have workbench.colorCustomizations or editor.tokenColorCustomizations, merge the new keys into those objects instead of replacing them.

### Save

Colors and token colors will apply right away.
